Данный проект является курсовым проектом по Теории алгоритмов и формальных языков на тему: "Построение аналитических моделей алгоритмов и оценка их сложности"

Главная задача при выполнении курсового проекта - реализация приложения с интерфейсом для работы с машинами Тьюринга, а именно необходимо реализовать алгоритм работы одноленточной и многоленточной машин Тьюринга для выявления принадлежности входного слова языку L = {aⁿb⁵ⁿc³ⁿ | n ≥ 1}
